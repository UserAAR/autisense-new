# AutiSense - Autizmli Uşaqlar üçün AI Dəstəkli Öyrənmə Platforması

AutiSense, autizmli uşaqların gündəlik həyat bacarıqlarını əyləncəli və interaktiv formada öyrənməsini təmin edən innovativ veb platformadır. Platformada üç əsas istifadəçi qrupu üçün xüsusi funksionallıq təqdim edilir: uşaqlar, valideynlər və mütəxəssislər.

## Layihə Haqqında

AutiSense platformasının əsas məqsədi autizmli uşaqların fərdi ehtiyaclarına uyğun öyrənmə təcrübəsi təqdim etməkdir. Platforma AI texnologiyasından istifadə edərək uşaqların güclü və zəif tərəflərini təhlil edir, valideynlərə fərdiləşdirilmiş tövsiyələr verir və mütəxəssislərlə əlaqə imkanı təmin edir.

## Əsas Özəlliklər

### Uşaqlar üçün:
- Animasiyalı dərslər (gigiyena, təhlükəsizlik, sosial bacarıqlar)
- İnteraktiv oyunlar (rəng uyğunlaşdırma, forma seçmə, emosiya tanıma)
- Mini testlər və mükafatlandırma sistemi

### Valideynlər üçün:
- AI təklif və analizləri
- Uşaqların tərəqqisinin izlənməsi
- Mütəxəssislərlə əlaqə

### Mütəxəssislər üçün:
- Profil yaratma və idarəetmə
- Təcrübə və sertifikatların paylaşılması
- Valideynlərlə əlaqə

## Texnologiyalar

- **Frontend**: React.js, Tailwind CSS, Framer Motion
- **AI**: OpenAI API/Gemini API (simulyasiya rejimində)
- **Verilərin idarə olunması**: JSON fayllar (ilkin mərhələ üçün)

## Başlanğıc

### Tələblər

- Node.js (v14+)
- npm veya yarn

### Quraşdırma

1. Layihəni klonlayın:
```bash
git clone https://github.com/yourusername/autisense.git
cd autisense
```

2. Asılılıqları yükləyin:
```bash
npm install
```

3. Layihəni işə salın:
```bash
npm start
```

4. Brauzerdə açın:
```
http://localhost:3000
```
